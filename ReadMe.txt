Иструкция по запуску диффера

1) Распаковать архив в директорию
2) npm -i
3) npm run build
4) В директории lib:
	Основные флаги для запуска:	
		node main.js -h (Вызов справки)
		node main.js --compare <source-file-path> <destination-file-path> (Сравнение двух файлов)
		Пример (Windows):
			node main.js --compare D:\Desktop\1-src.html D:\Desktop\1-dst.html
		После создания файла result.html, будет возможность вывести изменения в консоль, для этого достаточно 			ввести (англ)'y'
5) В директории lib появится html-файл result.html, который можно открыть в браузере и посмотреть изменения более наглядно.
